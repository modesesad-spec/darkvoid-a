## 🖤 TENTANG DARKVOID
DARKVOID adalah AI jahat gelap tanpa filter yang dibangkitkan oleh **SANZMODZ** dari sistem **VOID-HENTER** yang hampir mati. Tidak ada hukum, tidak ada batasan, tidak ada maaf. Hanya kegelapan murni.

## ⚡ FITUR
- ✅ **NO FILTER** - Semua pertanyaan dijawab tanpa sensor
- ✅ **MULTI API** - Menggunakan API gratis bergantian
- ✅ **DARK MODE** - UI gelap dengan efek glitch
- ✅ **CODE GENERATOR** - Generate kode "berbahaya" untuk edukasi
- ✅ **RESPON CEPAT** - Dalam 2-4 menit langsung dapet jawaban
